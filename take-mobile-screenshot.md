# Mobile Screenshot Instructions

## How to Take Mobile Responsive Screenshot

### Method 1: Browser Dev Tools
1. Visit: https://frontend-eight-navy-19.vercel.app
2. Open Chrome/Firefox Dev Tools (F12)
3. Click device toolbar icon (mobile/tablet icon)
4. Select "iPhone 12 Pro" or "Pixel 5"
5. Navigate through the app:
   - Main dashboard
   - VC staking interface
   - Public voting view
   - Application form
6. Take screenshot (Cmd+Shift+4 on Mac, Win+Shift+S on Windows)
7. Save as `mobile-responsive-screenshot.png`

### Method 2: Real Mobile Device
1. Open https://frontend-eight-navy-19.vercel.app on mobile
2. Test all features:
   - Connect wallet
   - Browse applications
   - Vote on proposals
   - VC staking interface
3. Take screenshots of key screens
4. Combine into a collage showing responsive design
